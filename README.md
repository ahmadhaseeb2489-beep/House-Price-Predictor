# AI House Price Predictor

A machine learning web application that predicts house prices based on various features using Linear Regression.

## Features
- **ML-Powered Predictions**: Accurate price estimates using Linear Regression
- **Beautiful Web Interface**: Responsive Flask web application
- **Real-time Results**: Instant predictions with detailed breakdown
- **Data Visualization**: Charts and analysis of housing data

## Technologies Used
- **Backend**: Python, Flask, scikit-learn
- **Frontend**: HTML5, CSS3, JavaScript
- **Machine Learning**: Linear Regression, pandas, numpy
- **Data Visualization**: matplotlib, seaborn

## Usage
- Open your browser to http://localhost:5000

- Enter house details:

- Size (sq ft)

- Number of bedrooms

- House age (years)

- Location type

- Get instant price prediction!
## Model Features
- Size: Price per square foot

- Bedrooms: Value per additional bedroom

- Age: Depreciation over time

- Location: Urban/Suburban/Rural pricing
